## Part 3. Настройка сети ОС

- Изменила название ВМ на user-1

- Вывела названия сетевых интерфейсов с помощью консольной команды ip -br link show. Интерфейс lo (loopback) — это виртуальный интерфейс для внутренней коммуникации системы с самой собой. Он позволяет сетевым приложениям и службам общаться локально без использования физической сети

## Part 8. Установка и базовая настройка сервиса SSHD

- Установила SSHd: sudo apt install openssh-server -y
- Добавила автостарт службы при загрузке системы: sudo systemctl enable ssh
- Изменила порт на 2022 в /etc/ssh/sshd_config, убрала значок #
- Проверила процесс: ps -ef | grep sshd
Объяснение ключей: -e все процессы, -f полный формат
- Перезагрузила систему: sudo reboot
- После перезагрузки установила net-tools: sudo apt install net-tools -y
- Проверила порт 2022: sudo netstat -tan. Вывод: tcp 0 0 0.0.0.0:2022 0.0.0.0:* LISTEN

**Скриншот c выводом команды netstat -tan**

![Скриншот c выводом команды netstat -tan](screenshot_part8.png)

-Объяснение команды netstat -tan:

Ключи:
- -t — отображение TCP-соединений
- -a — отображение всех соединений (включая слушающие порты)
- -n — отображение IP-адресов и портов в числовом формате

Столбцы вывода:
- Proto — протокол (tcp)
- Recv-Q — очередь полученных, но не прочитанных данных
- Send-Q — очередь отправленных, но не подтверждённых данных
- Local Address — локальный IP-адрес и порт
- Foreign Address — удалённый IP-адрес и порт
- State — состояние соединения (LISTEN — прослушивание)

Значение 0.0.0.0:
Означает, что служба SSHd слушает на всех сетевых интерфейсах системы, а не только на localhost (127.0.0.1). Это позволяет подключаться к серверу как с самой машины, так и по сети.
